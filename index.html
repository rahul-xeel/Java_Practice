CHARACTER CONVERSION TO UPPERCASE USING BIT
* Program to convert a lowercase alphabet into uppercase using bit manipulation.
* In ASCII, difference between 'a' (97) and 'A' (65) is the 5th bit (value 32).
* To make a character uppercase → clear the 5th bit using: result = ch & (~(1 << 5)).
* Example: 'a' = 0110 0001 → AND with 1101 1111 → 0100 0001 = 'A'.
* Works only for alphabets, already uppercase letters remain unchanged.
* Time complexity: O(1), since it is a single bitwise operation.
